Static route
Add route for a network ( This will only add route temporarly, add the command in '/etc/sysconfig/network-scripts/route-eth0' for permanent)

route add -net 10.0.0.0/8 gw 192.168.219.1

route add host 10.10.15.17 gw 192.168.219.2 #Add route for a host
delete route host
route del host 10.10.15.17

ip route # see routing table
route -n

Add route in windows server (persistent)
route -p ADD 192.168.35.0 MASK 255.255.255.0 192.168.0.2

vi /etc/network/interfaces


Restart network

/etc/init.d/networking restart

calculate total logical core

The total number of logical cores = CPU sockets × physical cores per socket × threads per physical core.
Therefore, the computer has 4 * 1 * 1 = 4 logical cores in total.
MYSQL database

Install maridb server

yum -y install mariadb mariadb-server

Once the installation is complete, start the MariaDB service and add it to the startup boot time.

systemctl start mariadb systemctl enable mariadb

Next, we’re going to configure the root password for the MariaDB using the interactive tool called ‘mysql_secure_installation’.

mysql_secure_installation

Login with username root

mysql -u root -p

If Access denied for user ‘root’@‘localhost’ (using password: YES) is promt , then use following command,

mysqld_safe --skip-grant-tables

Create database

create database [dbname];

Create a user ‘user1’, set password, assign it’s privileges to database ‘db_name’ and flush privileges

grant all privileges on db_name.* to user1@localhost identified by 'your_password'; flush privileges;
