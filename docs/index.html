Glassfish

Glassfish Server is a production ready Java EE 6 compliant application server developed under an open source license. Benefits include a modular, runtime that minimizes resource consumption, an easy-to-use administration console, a feature-rich command line tool, centralized administration and high-availability clustering.

NTP

NTP is an internet protocol that’s used to synchronise the clocks on computer networks to within a few milliseconds of universal coordinated time (UTC). Chronyd is a daemon for synchronisation of the system clock.

Redis

In-memory data structure store, used as database cache and message broker. In-memory database is a database that keeps the whole database in RAM. Supports data structure such as strings, hashes.

DKIM

Domain Key Identified Mail is email authentication technique that allows the receiver to check that an email was indeed sent and authorized by the owner of that domain. It is email security standard designed to make sure message weren’t altered in transist between the sending and receipient servers by the use of private and public key cryptography.

SPF

Sender Policy Framework (SPF) is authentication protocol used to indicate to mail exchanges which hosts are authorized to send mail for a domain. It is defined using TXT record type, also has SPF record type.

SNMP

Simple Network Management Protocol (SNMP) is a way for different devices on a network to share information helps in communication of devices of different hardware and software. It is a protocol used by network management tools to identify devices, monitor network performance, keep track of changes to the network, or determine the status of network devices in real time.

DMARC

Domain-based Message Authentication Reporting and conformance (DMARC) is designed to protect your company’s email domain for being used for email spoofing, phishing scans. It is enhancement to existing email authentication technologies.

Firewall

Firewalld provides a dynamically managed firewall with support for network/firewall zones to define the trust level of network connections or interfaces. firewalld provides support for zones, predefined services and ICMP types and has a separation of runtime and permanent configuration options.
