SNMP

Simple Network Management Protocol (SNMP) is a way for different devices on a network to share information helps in communication of devices of different hardware and software. It is a protocol used by network management tools to identify devices, monitor network performance, keep track of changes to the network, or determine the status of network devices in real time. It works on client-server model. Network manager collect and process information about devices on the network. Agents, are any type of device connected to the network

SNMP uses a tree-like format, under which data is always available for managers to collect. The data tree consists of multiple tables , which are called Management Information Bases, or MIBs. MIBs group together particular types of devices. MIB has a unique identifying number, as well as an identifying string. Individual devices in MIB has a unique Object Identifier, or OID

Versions

SNMPV1

SNMPv1 offers weak security features.

  1. Managers can authenticate to agents without encryption when requesting information.

  2. Anyone with access to the network could run “sniffing” software to intercept information about the network.

  3. Unauthorized device can easily pretend to be a legitimate manager when controlling the network.

  4. SNMPv1 uses certain default credentials which is rarely changed by admins

SNMPV2

  1. SNMPv2 offered some security enhancements

  2. But community access string to permit access to Simple Network Management Protocol (SNMP) via SNMPv1 still existed in SNMPv2

SNMPV3

SNMPv3 is the latest version of SNMP. Its features enhanced security.

SNMPv3 security was based on two factors

  1. Authentication

Credentials are required in order to authenticate the authorized manager to pull network information.

  1. Privacy

Encryption of SNMP message is done to ensure that it cannot be read by unauthorized users. Privacy is useful in applications where SNMP messages are routed over the Internet.

Installation of snmp v3 in linux

Install the required packages

yum install epel-release
yum install net-snmp net-snmp-utils net-snmp-devel

Configure SNMP version 3 user

net-snmp-config	--create-snmpv3-user [-ro] [-A authpass] [-X privpass]
                         [-a MD5|SHA] [-x DES|AES] [username]

Example

net-snmp-config --create-snmpv3-user -ro -A 'Xyz0@123$' -X abcd@1234 -x AES -a SHA snmpclient

Enable and start service

systemctl enable snmpd
systemctl start snmpd

Test the setup

snmpwalk -v3  -l authPriv -u snmpclient -a SHA -A 'Xyz0@123$'  -x AES -X 'abcd@123#' localhost
