Kaspersky for linux

In this tutorial, you will learn how to install Kaspersky in Centos Server and connect to Kaspersky Server.

Note : First telnet to the Kaspersky server port 13000 (default), It should success.

telnet Kapsersky_server_IP 13000

Step 1 : Install Required Dependencies:-

yum install 'perl(Data::Dumper)' -y
yum install perl-Digest-MD5 -y
yum install glibc -y
yum install kernel-devel -y

Step 2: Download and install Kaspersky Network Agent from its official site

rpm -ivh https://products.s.kaspersky-labs.com/endpoints/keslinux10/11.0.0.2706/multilanguage-INT-11.0.1.2706/3239363336377c44454c7c31/klnagent-12.0.1-45.i386.rpm

As it suggest, run postinstall.pl and specify Kaspersky server and other information.

Step 3: Install Kaspersky endpoint security application also from our local repository:-

rpm -ivh https://products.s.kaspersky-labs.com/endpoints/keslinux10/11.0.0.2706/multilanguage-INT-11.0.0.2706/3237363230337c44454c7c31/kesl-11.0.0-2706.i386.rpm

Again, as it suggests, run kesl-setup.pl script and fill the information.

With this KES and Netagent is installed and running:- Check the status:-

systemctl status klnagent64
systemctl status kesl-supervisor
Make it enable :-
Systemctl enable klnagent64
Systemctl enable kesl-supervisor

Step 4: Login to Kaspersky server. The device appears in unassigned devices.

Now, move it to the particular group to apply the key:-

Step 5: IF you see the status of the device, the real time protection is not enabled.

To make it effective Right click on the device > All task > Force Synchronization.

Step 6 : Add the key to the server.

step 7 : Restart the service if necessary.
